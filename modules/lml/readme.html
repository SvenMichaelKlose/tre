Overview

LML is XML stored as expressions. If an expression is not a text string, the first element of an expression must be the tag name as a symbol, followed by optional attributes as keyword arguments and then its children.

The tag and attribute names are converted to lower camel case.

Function $$ is used to convert LML into a DOM tree. This:

(princ ($$ `(html :lang "en" (head (title "Hello world!") (script :src "/main.js" :defer nil ; Attribute without value. "")) ; Ensure closing tag. (body "Hi there!"))))

will print

<html lang="en" <head> <title>Hello world!</title> <script src="/main" defer></script> </head> <body> Hi there! </body> </html>
